## Build Setup

``` bash
# install dependencies
$ npm run install

# Lint all .js, .ts, .vue files
$ npm run lint
# Lint and fix files
$ npm run lint -- --fix

# unit test with jest
$ npm test

# serve with hot reload at localhost:3000
$ npm run dev
```

For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
